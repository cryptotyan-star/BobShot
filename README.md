# BobShot

BobShot — простой скриншотер под macOS. Снимаешь область экрана, размечаешь поверх и сохраняешь или копируешь в буфер.

Нажал хоткей → экран **замирает** (стоп-кадр) → выделил область → разметил прямо поверх → скопировал или сохранил. Никакого отдельного окна редактора.

## Возможности

- **Стоп-кадр.** По хоткею снимается полный кадр каждого дисплея в этот момент — выделяешь по застывшему изображению, а не по «живому» экрану. Удобно для скриншотов по видео.
- **Глобальный хоткей** `⌃⌥S` (Ctrl+Option+S).
- **Разметка поверх скриншота**: перо, линия, стрелка, прямоугольник, маркер, текст, выбор цвета, undo.
- **Текст** многоцветный в одном блоке (цвет меняется для дальнейшего ввода), с переносом внутри выделения.
- **Палитра цвета** всплывает справа от свотча.
- **Копировать** в буфер (PNG+TIFF) или **Сохранить** PNG на Рабочий стол с авто-именем без замены (`BobShot.png` → `BobShot1.png` …).
- Корректный захват на Retina и мультимониторе (координатный маппинг выводится из реального размера снимка).

## Стек

Swift + AppKit, нативно под macOS 14+. Захват — ScreenCaptureKit (`SCScreenshotManager`); глобальный хоткей — Carbon. Сборка — SwiftPM (без полного Xcode, хватает Command Line Tools).

## Сборка

```bash
scripts/build-app.sh debug      # собрать build/BobShot.app
open build/BobShot.app
```

Скрипт собирает бинарь через `swift build`, упаковывает в `.app`, подписывает. Если в связке есть сертификат «BobShot Dev» (см. `scripts/make-signing-cert.sh`) — подпись стабильная, и разрешение «Запись экрана» переживает пересборки. Иначе ad-hoc.

## Установка (Launchpad)

```bash
scripts/install.sh              # собрать release + положить в /Applications
```

После этого BobShot появляется в Launchpad и Spotlight. Первый запуск из `/Applications` может потребовать заново выдать разрешение **Системные настройки → Конфиденциальность и безопасность → Запись экрана** (путь приложения изменился).

## Разрешения

Нужно одно: **Запись экрана** (для ScreenCaptureKit). Приложение само ведёт в нужный раздел настроек, если разрешения нет.

## Структура

```
Sources/BobShot/
├── main.swift                  ← точка входа
├── AppDelegate.swift           ← статус-бар, хоткей, стоп-кадр, экспорт (copy/save)
├── GlobalHotKey.swift          ← глобальный хоткей через Carbon
├── ScreenRecordingPermission.swift ← проверка/запрос TCC
├── ScreenCapturer.swift        ← захват дисплея + кроп региона из стоп-кадра
├── CoordinateMapper.swift      ← global pt → пиксели снимка (Retina/мультимонитор)
├── OverlayController.swift     ← borderless-оверлеи на всех экранах
├── SelectionView.swift         ← выделение + in-place редактор + панели
└── Annotations.swift           ← модели аннотаций + рендер/композит

scripts/   ← build-app.sh, install.sh, make-signing-cert.sh, convert-icons.sh
Resources/ ← Info.plist, иконки панели
plans/     ← технические планы (одна функция = один план)
```
